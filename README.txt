data文件夹下包含了实验所用到的所有数据，包含以下几个文件：
	①books_data.txt：每行为一本电子书的多维度数据，包含题名、作者、亚马逊评论和评分、微软学术主题标签、引用信息等。每行为一个字典对象，每个字典对象是对一本电子书的描述。
	②book_data_inMA_with_level1and2： 在books_data.txt中筛选出的能够在微软学术平台检索到且被标记了主题的电子书。请使用eval进行读取，读取后会获得一个列表对象，列表中的每个元素是一个字典对象，每个字典对象是对一本电子书的描述。
	③tags1_and_tags2_explains.xlsx：包含一级和二级主题在维基百科中的解释信息，共311个。
	④tags3_explains.txt：三级主题在维基百科中的解释信息，请使用eval进行读取，读取后会获得一个列表对象，列表中的每个元素是一个字符串，顺序与tag_3相同。
	⑤tag2.txt: 二级主题构成的列表，请使用eval进行读取。
	⑥tag2_3.txt:请使用eval进行读取，读取后会获得一个字典对象，字典的键是二级主题，值是该二级主题包含的三级主题所构成的列表。
